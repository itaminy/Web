Использованные команды

  • ssh [логин]@kubsu-dev.ru - подключение к серверу
  • ping kubsu.ru - проверка доступности и определение IP
  • nslookup kubsu.ru - получение A-записей
  • nslookup -type=MX kubsu.ru - получение MX-записей (если нужно)
  • whois kubsu.ru - информация о домене
  • git clone [репозиторий] - клонирование на сервер
